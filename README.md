# Calgary Traffic Incident Analysis (2021–2026)

This project analyzes traffic incidents in Calgary using data from 2021 to 2026.  
The goal is to identify high-risk areas, temporal patterns, and traffic hotspots.

---

## 📊 Visualizations

- **Interactive Heatmap**  
  Shows overall incident density across Calgary.

- **Hotspot Density Map (Hexbin)**  
  Highlights areas with the highest concentration of incidents.

- **Incidents by Hour**  
  Displays how incidents vary throughout the day.

- **Top Intersections**  
  Identifies the locations with the highest number of incidents.

---

## 📁 Files

- `incident_hotspots.py` → interactive heatmap (Folium)
- `hotspot_map.py` → hexbin density map (Matplotlib)
- `incidents_by_hour.py` → hourly trends
- `top_intersections.py` → top locations chart
- `Traffic_Incidents_filtered_2020.csv` → dataset
